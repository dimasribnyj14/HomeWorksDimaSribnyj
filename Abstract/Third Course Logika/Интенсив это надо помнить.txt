10.Вкажіть назву пунку меню в якому ми можемо переглянути всі швидкі клавіши

Работа с git (не путай с github)
27. Навіщо потрібна Система Контролю Версій?
Відповідь: Дозволяє зберігати кілька версій одного проекту,Дозволяє повертатися до попередніх версій
28. Оберіть схему за якою працює Git
Відповідь: Розробник 1 - локальний репозиторій - віддалений репозиторійРозробник 2 - локальний репозиторій- віддалений репозиторій
29. На що НЕ здатний Git?
Відповідь: Створення дерева(Tree)
30. Локальний репозиторій - це 
Відповідь: репозиторій, розташований на локальному комп'ютері розробника в каталозі.
31. Чим відрізняється репозиторій від гілки(Branch)?
Відповідь: Гілку можна об'єднати з іншою гілкою,Репозиторій містить в собі всі гілки
32. Що таке Комміт(Commit)?
Відповідь: Фіксація змін у локальний репозиторій
33. ... — надсилання всіх коммітів на віддалений сервер репозиторію.
Відповідь: Пуш (Push)
34. Чим відрізняється локальний репозиторій та віддалений репозиторій?
Відповідь: Локальний знаходиться на вашому комп'ютері, а віддалений на віддаленому комп'ютері
35. Які головні можливості Git?
Відповідь: Безкоштовний,Зручна система збереження,Простий спосіб створення гілок,Швидка робота
36. Чому потрібно працювати з Гілками(Branch)?
Відповідь: Дозволяють швидко та зручно внести змінни у проекті,Дозволяє паралельно працювати з вашою командою
37. З чого починається робота з Git Bash?
Відповідь: git init
38. ....- відображає стан робочого каталогу та розділу проіндексованих файлів.
Відповідь: git status 
39. Обрати правильний ланцюжок подій, щоб відправити файли на віддалений репозиторій
Відповідь: git add "FileName"git commit -m "CommitMessage"git push RemoteName BranchName
40. За допомогою якої команди можливо підключится до віддаленого репозиторію?
Відповідь: git remote
41. Що можна пушити до віддаленого репозиторію?
Відповідь: Гілку
42. Команда, яка дозволяє змінювати конфігурацію Git
Відповідь: git config
43. Чи можна НЕ записувати коментар до Commit на англійській мові?
Відповідь: Так
44. Яка команда дозволяє склонувати репозиторій?
Відповідь: git clone
45. ... - перераховує комміти, зроблені в репозиторії.
Відповідь: git log
46. Склонувавши репозиторій ви отримаєте
Відповідь: Всі гілки з репозиторію,Повню історію коммітів,Підлючення до віддаленого репозиторію
47. Що таке SSH-ключ?
Відповідь: це спеціальний код, який дозволяє віддаленому комп'ютеру зрозуміти хто ви є
48. За допомогою якої команди можно згенерувати SSH ключ?
Відповідь: ssh-keygen
