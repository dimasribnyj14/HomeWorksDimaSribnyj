1. Создать документ страницы с названием index, также файлы стилей и скрипта style, script.

2. Создать базовый шаблон страницы. Подключить стили и скрипт( незабываем, что подключаем его в конце тега body ).

3. Написать скрипт создающий HTML элементы с параметрами, которые передаются в аргументы функции.
4. Название функции createElement. Аргументы tag, properties, styles, parentElem. 
5. Первый аргумент, это с каким тегом создаётся элемент
6. Другие два аргумента функции это два списка, первый отвечающий за стандартные свойства HTML объекта и атрибутов, в данный список передаются списки с название свойства и его значением, второй отвевающий за стили, по такой же схема, передаётся список с названием свойства стиля и его значением.
7. Последним аргументом является родительский элемент в который добавляется созданный элемент. 

Пример параметров: 
createElement( tag = ‘input’, properties = [ [‘innerHTML’, ‘Lorem ipsum’], [‘type’, ‘text’], [‘placeholder’, ‘dolor sit amet’] ], styles = [ [‘fontSize’, ‘18px’], [‘color’, ‘red’] ], parentElem = document.querySelector('...'))
Вам нужно перебирать все принятые списки с свойствами и стилями. 0 - индексом является название свойства или стиля, первый индекс это значение.
Так как, свойства могут принимать множество значений, используйте перебор цикла.

for ( let propertie of properties ) {
    …
}
Также, но со стилями.
