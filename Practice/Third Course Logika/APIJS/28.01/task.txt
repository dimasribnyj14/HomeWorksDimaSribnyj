Задача на работу с API 

Объяснение кода:
Создает переменная url, которая хранит в себе ссылку на API. 
fetch - отправляет сетевые запросы на сервер — может как получать, так и отправлять данные. В нашем случае он получает данные NASA API.
Так как, запрос на сервер занимает время, в таком случае нам нужно чтобы код выполнился уже после полученных данных( в данном случае, порядок кода не имеет привязке по времени).
then это то что помогает в данной ситуации, он ждет пока слева от него код выполниться  и только тогда начинает сам выполнять свой код.
Если вы не понимаете как это работает, выполните alert после всех then и внутри последнего then.

Ваша задача:
1. Узнать что находится внутри полученных данных.
2. Создать элементы: заголовка, описания, картинки и даты, используя полученные данные. Все элементы создаются исключительно через js.
3. Закомментировать код.
4. Разобраться в работе с then и fetch. Написать свой код на основе данного. 
API взято с https://api.nasa.gov
