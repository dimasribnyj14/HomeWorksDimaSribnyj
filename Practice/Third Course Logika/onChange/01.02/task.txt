Задача на событие change.

Просмотрите файл index.html. 
Разберитесь с тегом select и option используя браузер.
В файле script.js функции changeColor, напишите строки которые изменяют цвет фона body.
Используйте атрибут value для option. В него запишите цвет за который отвечает option.
Пример:
<option value='Purple'>Purple</option>
Используя этот атрибут у option, Вам нужно обратится к ним в функции и изменить исходя из содержимого.

Напоминание!
Используется событие change!
В данной задаче используется всплытие с элемента option, на элемент select.
Всплытие – это когда на элементе происходит событие, обработчики сначала срабатывают на нем, потом на его отце, потом выше и так далее, вверх по цепочке предков.
